### 本周总结/感悟
```dataview
TABLE without ID
	file.link as 日期,
	L.text AS 每日总结
FROM "00Journal/01DailyNotes" 
WHERE date(date) >= date(this.date) - dur(6 days) AND date(date) <= date(this.date)
FLATTEN reverse(file.lists) AS L
WHERE meta(L.section).subpath = "今日总结"
SORT date(file.name) DESC
```
